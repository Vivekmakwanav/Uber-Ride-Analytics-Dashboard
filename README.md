# Uber Ride Analytics Dashboard

## Project Overview

This Power BI project analyzes Uber ride data to provide actionable business insights into booking trends, revenue performance, customer behavior, and trip efficiency.

## Objectives

- Analyze ride booking trends
- Monitor revenue and trip performance
- Identify customer behavior patterns
- Track key business KPIs
- Support data-driven decision-making

## Tools Used

- Power BI
- DAX
- Data Modeling
- Power Query
- Excel / CSV

## Key KPIs

- Total Bookings
- Total Revenue
- Average Trip Distance
- Average Trip Duration
- Revenue Trends
- Booking Status Analysis

## Dashboard Features

### Executive Summary
Provides a high-level overview of business performance.

### Revenue Analysis
Tracks revenue trends and growth patterns.

### Trip Analysis
Analyzes trip duration, distance, and booking behavior.

### Customer Insights
Identifies customer preferences and ride patterns.

## Dataset

The dataset contains ride information including:

- Booking ID
- Customer ID
- Pickup Location
- Drop Location
- Trip Distance
- Trip Duration
- Fare Amount
- Booking Status
